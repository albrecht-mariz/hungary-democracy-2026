# Is democracy falling in Hungary?

Hungary's Liberal Democracy Index dropped from **0.68 to 0.31** between 2010 and 2025 - the largest absolute decline of any country in the V-Dem dataset. V-Dem classifies it as an electoral autocracy since 2018, the only EU member state with that label. 

Elections on **12 April 2026**.

Interactive visualisation built with React and D3, tracking 179 countries over 15 years.
